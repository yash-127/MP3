🎵 Advanced MP3 Streaming Platform

📖 Overview
This is a feature-rich, full-stack MP3 streaming platform and one of the most advanced web projects I have built from the ground up. It features a complete user-facing music player interface and a secure, dedicated admin portal for content management. Designed for a seamless audio experience, users can discover music, curate their personal playlists, and save their favorite tracks.

✨ Key Features

🎧 User Experience
Dynamic Audio Player: Smooth playback controls for an uninterrupted listening experience.
Interactive Like System: Users can easily "heart" or like their favorite tracks and albums.
Personalized Libraries: Dedicated pages for "Liked Songs" and "Liked Albums" so users can always find their favorites.
Custom Playlists: Users can create, build, and manage their own custom playlists.
Discover & Browse: Explore the library through categories, albums, and a randomized discovery feed.
User Authentication: Secure sign-up, login, and session management.

🛡️ Admin Portal
Content Management: Dedicated admin dashboard to upload and manage new audio files.
Library Organization: Create, edit, and organize albums and music categories.
Platform Oversight: Complete control over the platform's music database directly from the backend.

🛠️ Tech Stack
This project was built entirely without external frameworks to ensure deep control over the architecture and logic.
Frontend: HTML5, CSS3, JavaScript
Backend: PHP
Database: MySQL

🚀 Installation & Local Setup
Clone the Repository:
git clone https://github.com/yourusername/your-repo-name.git
Move to Local Server:
Place the extracted project folder into your local server's web directory (e.g., www for EasyPHP/WAMP, or htdocs for XAMPP).

Database Setup:
Open phpMyAdmin (or your preferred MySQL manager).
Create a new database for the project.
Import the project's SQL file to structure the tables for users, songs, playlists, and likes.

Configure Connection:
Update the database connection variables (username, password, database name) in your PHP configuration file to match your local environment.
Run the Project:
Open your web browser and navigate to http://localhost/MP3 (or your specific directory name).
